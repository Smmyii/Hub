# Keybindings

All keyboard shortcuts across the system.

---

## Hyprland (Window Manager)

### Shell / UI Panels

| Shortcut | Action |
|----------|--------|
| `Super` (tap) | Toggle overview/launcher |
| `Super + V` | Clipboard history |
| `Super + .` | Emoji picker |
| `Super + A` | Toggle left sidebar |
| `Super + N` | Toggle right sidebar |
| `Super + P` | Toggle cheatsheet |
| `Super + K` | Toggle on-screen keyboard |
| `Super + M` | Toggle media controls |
| `Super + J` | Toggle bar |
| `Ctrl + Alt + Delete` | Session menu |
| `Ctrl + Super + T` | Wallpaper selector |
| `Ctrl + Super + Alt + T` | Random wallpaper |
| `Ctrl + Super + R` | Restart widgets |
| `Super + Alt + W` | Cycle panel family |

### Screenshots / Recording / OCR

| Shortcut | Action |
|----------|--------|
| `Super + Shift + S` | Screen snip (region) |
| `Print` | Fullscreen screenshot to clipboard |
| `Ctrl + Print` | Fullscreen screenshot to clipboard + file |
| `Super + Shift + A` | Google Lens (region search) |
| `Super + Shift + X` | OCR (text recognition to clipboard) |
| `Super + Shift + C` | Color picker (hex to clipboard) |
| `Super + Shift + R` | Record region (no sound) |
| `Super + Shift + Alt + R` | Record screen (with sound) |

### Window Management

| Shortcut | Action |
|----------|--------|
| `Super + Q` | Close window |
| `Alt + F4` | Close window (Windows-style) |
| `Super + Shift + Alt + Q` | Force kill window |
| `Super + Arrow Keys` | Focus window in direction |
| `Super + Shift + Arrow Keys` | Move window in direction |
| `Super + Mouse Drag (left)` | Move window |
| `Super + Mouse Drag (right)` | Resize window |
| `Super + Alt + Space` | Toggle float/tile |
| `Super + D` | Maximize |
| `Super + F` | Fullscreen |
| `Super + ;` / `Super + '` | Adjust split ratio |

### Workspaces

| Shortcut | Action |
|----------|--------|
| `Super + 1-0` | Switch to workspace 1-10 |
| `Super + Tab + 1-0` | Send window to workspace 1-10 (silent) |
| `Ctrl + Super + Left/Right` | Previous/next workspace |
| `Super + Page Up/Down` | Previous/next workspace |
| `Super + Scroll Up/Down` | Previous/next workspace |
| `Super + Shift + Page Up/Down` | Send window to prev/next workspace |
| `Super + S` | Toggle scratchpad |
| `Super + Alt + S` | Send window to scratchpad |
| `Ctrl + Super + Up/Down` | Jump 5 workspaces up/down |

### Media

| Shortcut | Action |
|----------|--------|
| `Super + Shift + N` | Next track |
| `Super + Shift + B` | Previous track |
| `Super + Shift + P` | Play/pause |
| `Super + Shift + M` | Toggle mute |
| `Super + Alt + M` | Toggle mic mute |
| `XF86Audio*` | Hardware media keys (volume, play, etc.) |
| `XF86MonBrightness*` | Hardware brightness keys |

### Screen / Zoom

| Shortcut | Action |
|----------|--------|
| `Super + =` | Zoom in |
| `Super + -` | Zoom out |

### Apps

| Shortcut | Action |
|----------|--------|
| `Super + Return` | Terminal |
| `Super + T` | Terminal (alt) |
| `Super + E` | File manager |
| `Super + W` | Browser |
| `Super + C` | Code editor |
| `Super + X` | Text editor |
| `Ctrl + Super + V` | Volume mixer |
| `Super + I` | Settings |
| `Ctrl + Shift + Escape` | Task manager |

### Session

| Shortcut | Action |
|----------|--------|
| `Super + L` | Lock screen |
| `Super + Shift + L` | Suspend / sleep |
| `Ctrl + Shift + Alt + Super + Delete` | Shutdown |

### Virtual Machine Mode

| Shortcut | Action |
|----------|--------|
| `Super + Alt + F1` | Enter VM submap (disables all keybinds) |
| `Super + Alt + F1` (in VM mode) | Exit VM submap |

### Sticky Notes

| Shortcut | Action |
|----------|--------|
| `Super + Shift + G` | Toggle sticky notes panel |
| `Super + Shift + N` | Create new sticky note |

## Kitty (Terminal)

### Tabs

| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + T` | New tab (keeps current directory) |
| `Ctrl + Shift + W` | Close tab |
| `Ctrl + Tab` | Next tab |
| `Ctrl + Shift + Tab` | Previous tab |
| `Ctrl + 1-5` / `Alt + 1-5` | Jump to tab 1-5 |
| `Ctrl + Shift + .` | Move tab forward |
| `Ctrl + Shift + ,` | Move tab backward |
| `Ctrl + Shift + Alt + T` | Rename tab |

### Splits

| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + Enter` | New split (auto) |
| `Ctrl + Shift + -` | Horizontal split |
| `Ctrl + Shift + \` | Vertical split |
| `Ctrl + Shift + Arrow Keys` | Navigate between splits |
| `Ctrl + Alt + Arrow Keys` | Resize splits |
| `Ctrl + Shift + Z` | Toggle zoom (stack layout) |

### General

| Shortcut | Action |
|----------|--------|
| `Ctrl + C` | Copy / interrupt |
| `Ctrl + F` | Search |
| `Page Up / Down` | Scroll |
| `Ctrl + +` / `Ctrl + -` | Zoom in / out |
| `Ctrl + 0` | Reset zoom |
